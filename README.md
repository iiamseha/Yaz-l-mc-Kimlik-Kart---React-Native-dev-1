# 📱 Yazılımcı Kimlik Kartı — React Native Ödev 1

## 📋 Proje Hakkında
React Native ile geliştirilmiş basit bir kimlik kartı uygulaması.
Props ve State kavramlarını kullanarak "İşe Al" butonuyla
geliştiricinin müsaitlik durumunu değiştiriyor.

## 🧩 Kullanılan Kavramlar
- **Props:** `ad`, `uzmanlik`, `seviye`
- **State:** `musaitMi` (boolean)
- **StyleSheet** ile Flexbox tabanlı modern kart tasarımı
